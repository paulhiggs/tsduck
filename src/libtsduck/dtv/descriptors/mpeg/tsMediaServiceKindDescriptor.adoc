==== media_authenticity_descriptor

Defined by MPEG in <<ISO-13818-1>>.

[source,xml]
----
<media_authenticity_descriptor
    uuid="string, optional"
    uri="string, optional"
    register_index="uint10, optional">

    <authenticated_stream
        stream_type="uint8, optional"
        authenticated_stream_ids="string"/>
        <!-- authenticaed_stream_ids should be an XML list if up to 32 unsigned byte values-->

</media_authenticity_descriptor>
----
